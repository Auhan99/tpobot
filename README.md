# TPO IIT (BHU) Bot

An utterly fantastic messenger bot to give updates of Traning and Placement portal of IIT BHU.

## Installation

  * `sudo apt-get install rabbitmq-server`
  * `pip install -r requirements.txt`
  *  make sure redis is >= 2.10.5
